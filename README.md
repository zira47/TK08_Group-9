# Simple C Calculator

## Project Description
This project is a simple command-line calculator written in C. It was developed as a collaborative team project to practice Git workflow, branching, and code reviews. The calculator accepts user input and performs basic arithmetic operations.

## Features
The calculator currently supports the following mathematical functions:
* **Add (Addition):** Computes the sum of two numbers.
* **Sub (Subtraction):** Computes the difference between two numbers.
* **Mul (Multiplication):** Computes the product of two numbers.
* **Div (Division):** Computes the quotient of two numbers (includes zero-division handling).
